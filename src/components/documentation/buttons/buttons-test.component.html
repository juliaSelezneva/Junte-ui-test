Buttons

Used to activate operations

Requirements

  1. The button must have different sizes: tiny, small, normal, large
  2. The button may have a badge and an icon
  3. The button should show the loading
  4. The button should respond to mouse hover, click
  5. The button should change the display when it is disabled
  6. The button must have a design in accordance with a given color scheme
  7. The button should be of two types: outline and fill
  8. The button should be able to stretch across the width of the container or be the width of the content
