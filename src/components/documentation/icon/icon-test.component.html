Icon

Icon is standard UI element presented in all frameworks. Icons can be implemented by fonts or SVG technology. We support both.

Requirements

  1. Icon must support multiple icon sets.
  2. Icon must support font and SVG glyphs.
  3. Icon must have mock preview.

Font icon

To create custom icon fonts you need install Gulp and add new command to package.json

After using this command will creating iconfont folder with config file (iconfont.json) and template of output scss font (template)

You need edit iconfont.json file for provide paths to your icons, output font and font name

After editing config file you need reuse "iconfont" command and will get in "src/assets/fonts/custom/custom-font.scss".

For implementing your iconfont you need add it to angular.json / architect / build / options / styles
